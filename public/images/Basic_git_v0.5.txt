1. git pull: lấy code mới nhất về
2. git push: đẩy code lên từ nhánh hiện tại
3. git branch -a: Xem tất cả branch 
4. git branch <branch-name>: để tạo nhánh branch mới từ branch hiện tại
5. git checkout <branch-name>: để chuyển qua branch khác
6. git add {file(regex)} : Thêm file vào change state / Nếu file = . => thêm tất cả file / file có thể là *.js => thêm tất cả file .js
7. git commit -m "Message": Commit code với message
8. git restore . : Xoá tất cả thay đổi code 
9. git help {command_name}: mở doc của git để đọc
10. git status: Xem trạng thái các file đã update || commit
11. git log: check tất cả commit 
12. git fetch {branch_name}: Cập nhật lại trạng thái của {branch_name}
  - git fetch -p : Cập nhật lại trạng thái của tất cả các branch 
13. git switch {branch}: Chuyển branch + pull code
